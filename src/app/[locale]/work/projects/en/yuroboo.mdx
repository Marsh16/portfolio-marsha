## Overview

Yuroboo is a charming static catalog website designed for a client specializing in handmade crochet accessories for popular vinyl toys including Labubu, Smiski, and Hirono. The website serves as a digital storefront showcasing the artisan's beautiful handcrafted pieces, each made with love and attention to detail.

Built with Next.js and styled with Tailwind CSS, the site provides an elegant and responsive platform for customers to browse the extensive collection of crochet accessories, learn about custom orders, and connect with the creator's passion for handmade craftsmanship.

## Technologies Used

- **Static Site Generation**: Next.js for optimized performance and SEO-friendly static pages
- **Responsive Design**: Tailwind CSS for beautiful, mobile-first responsive layouts
- **Image Optimization**: Next.js Image component for fast loading product galleries
- **Performance**: Static generation for lightning-fast page loads and excellent user experience
- **Accessibility**: Semantic HTML and ARIA labels for inclusive design

## Challenges and Learnings

Building Yuroboo required balancing aesthetic appeal with functionality while maintaining the handmade, artisanal feel of the brand. The challenge was creating a modern web presence that still conveyed the personal touch and craftsmanship of handmade products.

Key considerations included:
- Optimizing product images while maintaining quality for detailed crochet work
- Creating an intuitive navigation system for browsing different accessory categories
- Designing custom order forms that capture customer requirements effectively
- Ensuring the site reflects the warm, personal nature of the handmade business

## Outcome

Yuroboo successfully provides the client with a professional digital storefront that beautifully showcases their handmade crochet accessories. The static website delivers excellent performance, SEO benefits, and a delightful user experience that helps customers discover and appreciate the artisan's craftsmanship.
