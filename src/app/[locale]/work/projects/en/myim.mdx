## Overview

MyIM is a comprehensive meme marketplace that allows users to discover, purchase, and create custom WhatsApp stickers from a curated collection of memes. The platform combines e-commerce functionality with creative tools, enabling users to browse premium memes, purchase them using a credit system, and transform them into personalized WhatsApp sticker packs.

Built with modern web technologies, MyIM features seamless payment integration, advanced canvas-based editing, and a complete admin dashboard for content management.

## Key Features

- **Meme Marketplace**: Browse and purchase premium memes with rarity system (common, rare, epic, legendary)
- **WhatsApp Sticker Creator**: Advanced canvas-based sticker editor with customization tools
- **Payment Integration**: Xendit payment gateway with automated processing via n8n workflows
- **Sticker Pack Creator**: Create custom sticker packs with up to 30 stickers
- **Secure Authentication**: Email OTP authentication powered by Supabase
- **Admin Dashboard**: Complete content management system for meme uploads and user management
- **Credit System**: Virtual currency system for seamless purchases
- **Transaction History**: Complete billing and payment tracking with real-time updates
- **PWA Support**: Installable progressive web app experience

## Technologies Used

- **Frontend**: React 18 with TypeScript for type-safe development
- **Styling**: Tailwind CSS for responsive, utility-first design
- **Build Tool**: Vite for fast development and optimized builds
- **Canvas API**: Advanced image manipulation for sticker creation
- **Database**: Supabase PostgreSQL with Row Level Security (RLS)
- **Payment**: Xendit Indonesian payment gateway integration
- **Automation**: n8n workflow automation as backend service
- **Icons**: Lucide React for beautiful, consistent iconography

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Backend**: Supabase, n8n, PostgreSQL
- **Payment**: Xendit Payment Gateway
- **Authentication**: Supabase Auth with Email OTP
- **Database**: PostgreSQL with RLS, stored procedures, and triggers
- **Deployment**: PWA-enabled web application

## Challenges and Learnings

Building MyIM presented several technical challenges, particularly in integrating multiple payment systems with real-time database updates. The canvas-based sticker editor required deep understanding of HTML5 Canvas API for image manipulation and export functionality.

Key technical achievements:
- Implementing webhook-driven architecture with n8n for payment processing
- Building a sophisticated canvas editor for WhatsApp sticker creation
- Designing a scalable database schema with automated balance management
- Creating a seamless PWA experience with offline capabilities
- Integrating Xendit payment gateway with real-time transaction updates

The project showcased advanced full-stack development skills, from complex frontend interactions to automated backend workflows.

## Outcome

MyIM successfully demonstrates the integration of e-commerce, creative tools, and payment systems in a single platform. The application provides users with an intuitive way to discover and purchase memes while offering powerful tools to create personalized WhatsApp stickers.

The platform's architecture, combining React frontend with Supabase backend and n8n automation, creates a scalable and maintainable solution that handles real-time payments, user authentication, and content management efficiently.

This project highlights expertise in modern web development, payment integration, and creating user-centric applications that solve real-world problems in the digital content space.
